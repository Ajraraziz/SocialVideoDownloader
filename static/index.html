API Documentation

GET /api/info

Obtient les informations d'une vidéo

curl -X GET "/api/info?url=https://youtube.com/watch?v=..."

POST /api/download

Démarre le téléchargement d'une vidéo

curl -X POST "/api/download" -d '{"url": "...", "quality": "best"}'

GET /api/supported-sites

Liste des sites supportés

curl -X GET "/api/supported-sites"
